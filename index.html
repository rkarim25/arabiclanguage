      Level 1 — الأَسَاس (Foundation)

      Present tense, daily life, family. Upper-beginner: assumes you can read the Arabic script.

      How to use each story

      1. اِسْتَمِعْ Listen — play the story before reading. Listen twice; catch what you can.
      2. اِقْرَأْ Read — read with vowels on, tap any word you don't know. Then turn vowels off and read again.
      3. اِحْفَظْ Memorize — grade yourself on each flashcard honestly; missed words return sooner.
      4. أَجِبْ Quiz — answer the comprehension questions without looking back.
      5. تَكَلَّمْ Speak — say every sentence aloud; the microphone checks how close you were.
      6. اُكْتُبْ Write — dictation and translation. Writing is what locks vocabulary in.

      Come back daily and clear the Review queue — that's where memorization actually happens.

      ☁ Cloud sync & coaching

      Connect once and the site syncs your learning data to your private GitHub repo (arabic-learning-data). Your coach analyzes it and posts notes back here.

      How to connect (one-time, ~2 minutes)
      1. Open github.com/settings/personal-access-tokens/new (sign in as rkarim25).
      2. Token name: arabic-site-sync. Expiration: 1 year.
      3. Repository access → Only select repositories → choose arabic-learning-data.
      4. Permissions → Repository permissions → Contents: Read and write.
      5. Generate, copy the token (starts with github_pat_), paste it below.
